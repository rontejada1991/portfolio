Armadora Gold Setup

This small web application is used as a companion for the physical board game titled Armadora by Blue Orange Games. The app replicates the board used in the physical game and randomly populates the pretermined cells with a set of numbers provided by the game. The app may also be reset without refrehsing the page if users choose to start a new game.

Front End: HTML, CSS, JavaScript, JQuery, Semantic UI

Investigator's Handbook

The Investigator's Handbook is a web application designed to accompany the physical board game titled Elder Sign by Fantasy Flight Games. This app keeps track of up to 8 character objects (the maximum allowed in the game), has a dropdown menu to selet the character the user wishes to use, as well as an option to choose one randomly. The app also keeps track of the characters that have been picked, preventing them from being picked again within the same game session. There is also a reset button for users who wish to start a new game without needing to refresh the page.

Front End: HTML, CSS, JavaScript, JQuery, Bootstrap, Google Fonts, Font Awesome

Combinations & Permutations

Combinations & Permutations is a simple tool designed to solve combination and permutation problems.

Designed to be quick and intuitive for those who need a quick answer to their mathematical problems.

The app asks simple questions that it will use to produce an answer for you. These questions are designed to help you think critically while also taking care of the math for you. If you don't know whether the problem is a combination or a permutation, the application takes care of that for you as well.

Technologies Used: Java, XML, Android API, Java API, Android SDK

Under Bubble

Under Bubble is a fixed shooter arcade game designed for those who want a quick and fun experience.

Gameplay features include -

  • Distinct color variations for the player's avatar
  • Contrasting backgrounds ranging from light to dark atmospheres
  • Different types of enemies, each with their own speed and trajectory behaviors
  • Unique power-ups that can be utilized in a variety of different maneuvers

Other features include -

  • High Scores are saved to set a goal to beat the next time you play or to be the best among your friends and family!
  • Other statistics are recorded for you so you can keep track of your progress
  • Supports mute sound, for those who want to play while listening to their favorite music
  • More features will be added based on player feedback!

Technologies Used: Java, LibGDX Framework, Universal Tween Engine, Java API, Android SDK

Card Game Helper

Card Game Helper is designed to be a universal tool for card game players who need one app for all of their card games needs.

Universal Support - Popular supported card games include Magic: The Gathering, Yu-Gi-Oh!, Pokemon, Naruto Shippuden and many others. If the card game requires you to keep track of player scores, use dice rolls, coin flips, or counters, then Card Game Helper supports it!

Personal - Players can enter their names on top of their scores to make battles even more personal.

Backtracking - Previous dice rolls and coin flips are recorded for convenience in case players need to backtrack.

Sound - Battles come to life as sound effects can be heard whenever damage is dealt or recovered, a game is restarted, counters are placed or whenever coins are flipped and dice are rolled.

Technologies Used: Java, XML, Android API, Java API, Android SDK
